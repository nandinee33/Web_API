FAQ

How to get an API key?

1. Sign up and get an API key (APPID) on your account page. 2.After verification, find our welcome email that contains your API key and information.

Do you offer student discounts?

we don't offer student discounts. Please use the free plan. From time to time, we offer various promotions, so keep an eye out for those!

How many keys can I create?

You can create as many keys as you need. The total number of API calls is accumulated from all the API keys that belong to your account.

What is the format of the API response?

The JSON format is used by default. To get data in the XML or HTML format, just use “xml” or “html” for the mode parameter.

Why are some parameters missing in the API response?

Only the actually measured or calculated data is displayed in an API response.

About

Web API (Application Programming Interface) is used for helping the software components with each other. ASP.NET Web API is a framework that makes it easy to build HTTP Services that reach a broad range of clients, including browsers and mobile devices. Using ASP.NET, web API can communicate with different devices from the same database.In Web API, we use default action methods. Get, Post, Put, and Delete are four default action methods. When we add new Web API empty controller in our project, the above mentioned action methods are added by default along with some overloaded methods.
